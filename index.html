Enter, Packary

Simple

Create a repository in a few clicks, configure your package manager, and push your code. No hosting, proxy'ing and minimal configuration.

Multiple Languages

Initial support for Maven - NPM, Python plus other languages driven by community needs are planned

Made for teams

Each developer gets their own account, and you control who can publish what.

Where we're up to

We're on the verge of opening an invitation-based beta with support for Maven repositories, after that we're looking to involve the community on what to deliver next! The platform has been built with other languages in mind, so anything's possible!

A high-level road map includes (in no particular order)

  • Maven snapshots
  • NPM support
  • Python support

Pricing

We're still crunching numbers so aren't able to publish a pricing plan just quite yet.

As easy as 1, 2, 3.

  1. Choose your language and deploy a registry in one-click. Your first registry is free.

  2. Use our simple docs to configure your package manager. Chunky config blocks are ready to copy-paste.

  3. Our registries use the same APIs as their open source counterparts, so they'll work with all your existing tools/CI processes.
